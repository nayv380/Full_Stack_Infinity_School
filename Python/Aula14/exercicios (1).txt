Ex02. Você deverá implementar uma calculadora de impostos para pessoa juridica, seu dever será
utilizar a classe base abaixo para implementar 3 tipos de impostos:

```
class Imposto:
    def calcular(self, valor: float):
        return 0
```

a) Implemente a classe "DAS" que deve retornar 6% do valor informado
b) Implemente a classe "ISS" que deve retornar 5% do valor informado
c) Implemente a classe "IRPJ" que deve retornar 10% do valor informado
d) Ao final implemente uma funcao chamada "calcular_impostos" que irá receber um "valor" (float) e os "impostos" (list[Imposto]), essa função deve deduzir todos os impostos da lista do valor informado.

Exemplo de Utilização

```
def calcular_impostos(valor: float, impostos: list[Imposto]) -> float:
    ...

valor_bruto = 8000
impostos = [DAS(), ISS()]

# valor_bruto - 11% (Das + Iss)
valor_liquido = calcular_impostos(valor_bruto, impostos)
print(valor_liquido)
```
